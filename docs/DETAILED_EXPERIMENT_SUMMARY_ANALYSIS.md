# Ultra-LSNT 风电功率预测实验详细拆解总结

## 项目概述

**Ultra-LSNT（Ultra-Lightweight Sparse Neural Transformer）** 是一个针对风电功率预测任务设计的深度学习模型。项目包含全面的实验验证体系，涵盖性能对比、多领域泛化、鲁棒性测试、消融研究等多个维度。

**项目时间线**：2025-01-22 至 2026-01-25
**硬件环境**：RTX 4090D 24GB, 256 CPU核心, 755GB内存
**软件栈**：Python 3.10+, PyTorch 2.1.2, CUDA 11.8

### 1.2 数据预处理流程
1. **数据加载**：CSV文件读取，智能列检测
2. **缺失值处理**：线性插值 + 前后向填充
3. **特征选择**：删除非数值列、时间戳列、重复列
4. **标准化**：StandardScaler（仅用训练集拟合）
5. **序列构造**：seq_len=96 → pred_len=24（15分钟间隔）

### 1.3 评估指标体系

#### 核心指标
- **R² (决定系数)**：主要性能指标，越高越好
- **RMSE (均方根误差)**：绝对误差度量
- **MAE (平均绝对误差)**：稳健误差度量
- **MAPE (%)**：相对百分比误差
- **MSE (均方误差)**：平方误差

#### 概率预测指标（部分实验）
- **CRPS (连续分级概率评分)**：概率分布准确性
- **PICP (预测区间覆盖率)**：区间覆盖真实值的比例
- **MPIW (平均预测区间宽度)**：区间宽度度量
- **Pinball损失**：分位数回归损失

#### 工程指标
- **计算比率**：MoE激活专家比例
- **跳跃率**：跳过计算的序列比例
- **慢速模式比率**：使用深度计算路径的比例

## 第二部分：模型架构与方法

### 2.1 Ultra-LSNT 核心架构

#### 核心组件
1. **时序编码器 (TemporalEncoder)**：
   - 输入：`(batch, seq_len, input_dim)`
   - 特征投影 + 位置编码
   - 时序卷积（局部模式捕获）
   - 时序注意力（全局依赖）
   - 序列聚合为固定维度

2. **LSNT块 (UltraLSNTBlock)**：
   - **稀疏MoE路由器**：8专家，top-4激活，异方差感知
   - **双模传播**：快速路径（线性） + 慢速路径（深度MLP）
   - **一致性跳跃门控**：动态跳过不必要计算
   - **层归一化 + Dropout**

3. **时序解码器 (TemporalDecoder)**：
   - 固定维度展开为预测序列
   - 预测头（单变量或多变量输出）

#### 关键技术特性
- **混合精度训练**：AMP自动混合精度
- **温度退火**：router softmax温度从2.0降至0.1
- **路由器正则化**：Z-loss + 负载均衡loss
- **GPU优化**：大batch_size(256)，cuDNN基准优化

### 2.2 基线模型对比体系

#### 深度学习基线
1. **DLinear**：线性分解模型（最佳性能基线）
2. **Transformer**：标准自注意力架构
3. **Informer**：ProbSparse注意力 + 蒸馏
4. **Autoformer**：自相关 + 分解架构
5. **LSTM**：传统循环神经网络
6. **MLP**：多层感知器

#### 传统机器学习基线
1. **Persistence**：持续性预测
2. **Seasonal Naive**：季节性朴素预测
3. **RandomForest**：随机森林回归
4. **PowerCurve**：物理功率曲线方法

#### 梯度提升树
1. **LightGBM**：轻量级梯度提升机
